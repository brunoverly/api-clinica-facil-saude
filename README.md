## 👥 Sobre o projeto

O **ClinicaFácil** é uma API desenvolvida para o gerenciamento seguro e consistente de agendamentos de consultas médicas. O foco principal do sistema é garantir as regras de negócio críticas e integridade de agenda, como:
- Impedir que um médico possua agendamentos conflitantes no mesmo horário.
- Bloquear agendamentos fora do horário comercial da clínica.
- Permitir cancelamentos apenas com pelo menos 24 horas de antecedência.
